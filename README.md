# blog-vue

基于Vite创建的Vue3+Ts项目

## 推荐的IDE配置

[VSCode](https://code.visualstudio.com/) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar) (禁用 Vetur).

## 项目设置

```sh
npm install
```

### 开发时编译和热重载

```sh
npm run dev
```

### 为生产进行类型检查、编译和压缩

```sh
npm run build
```
